# Multi-Exam Eligibility Evaluation System (C)

This is a rule-based C program that checks eligibility for major Indian exams
like UPSC, JEE Main, GATE, and NEET based on user input.

## Exams Supported
- Class 10
- Class 12
- Graduation
- JEE Main
- UPSC
- GATE
- NEET

## Note
This project is for **academic learning purposes only**.
Eligibility rules are simplified and not official.

## How to Run
gcc main.c eligibility_rules.c -o eligibility  
./eligibility
